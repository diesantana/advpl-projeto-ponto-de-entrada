# 🧾 Validação com Ponto de Entrada MT100LOK (ADVPL)

Projeto desenvolvido em ADVPL com foco em estudos de Pontos de Entrada (PE) no Protheus e implementação de regras de negócio durante o processo de entrada de notas fiscais.

## 📌 Sobre o projeto

O projeto consiste em uma validação utilizando o ponto de entrada `MT100LOK` para impedir a inclusão de itens em documentos de entrada quando a quantidade for maior ou igual a 5 e a TES utilizada não estiver previamente permitida.

As TES válidas são configuradas dinamicamente através do parâmetro `MV_ZZTESQT`, evitando regras fixas no código.

Este projeto foi desenvolvido com foco em estudos de customização no Protheus, manipulação de GRID e validações de negócio em ADVPL.
